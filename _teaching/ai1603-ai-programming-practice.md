Course Information
======

- Course code: AI1603
- Credits: 2
- Offering unit: School of Computer Science

Course Content
======

This course introduces Python programming foundations, common tools for data processing and visualization, supervised and unsupervised learning, regression, neural networks, and selected artificial intelligence applications. It combines programming fundamentals with hands-on data mining and machine learning practice.

Assessment
======

Assessment consists of quizzes, homework assignments, and a final project. The project emphasizes practical problem solving and involves data analysis, model development, experimentation, and presentation.
