Course Information
======

- Course code: CS0501
- Credits: 3
- Offering unit: School of Computer Science

Course Content
======

This course covers core topics including linear lists, stacks and queues, trees, priority queues, search structures, hash tables, and graphs. It emphasizes both conceptual understanding and implementation, helping students connect abstract data structure design with concrete algorithmic coding practice.

Assessment
======

Assessment consists of programming assignments and a final examination. The course is designed to evaluate both understanding of fundamental concepts and the ability to implement core data structures and algorithms in code.
