Course Information
======

- Course code: CS4304
- Credits: 3
- Offering unit: School of Computer Science
- Co-instructors: Weiwen Liu, Guanjie Zheng

Course Content
======

The course includes the following seven major topics:

- MapReduce and Spark
- Hashing
- Large-Scale Machine Learning
- Dimensionality Reduction
- Graph Mining
- Recommendation System
- Time Series Data Mining

It covers foundational themes in large-scale data processing and algorithms while also incorporating implementation-oriented demos and project-based training.

Assessment
======

Assessment consists of a survey paper, programming homework, and a final course project with presentation. The course is designed to develop reading ability, implementation skills, and project communication in the area of big data algorithms.
